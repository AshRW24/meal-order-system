# 外卖订餐系统 - 用户端

基于 Vue 3 + Element Plus 的用户端系统

## 技术栈

- Node.js: 22.20.0
- npm: 10.9.3
- Vue 3: 最新版
- Element Plus: 最新版
- Vue Router: 4.x
- Pinia: 最新版
- Axios: 最新版
- Vite: 最新版

**详细环境配置**：参见 `../docs/环境配置.md`

## 安装依赖

```bash
npm install
```

## 运行项目

```bash
npm run dev
```

访问：http://localhost:5174
